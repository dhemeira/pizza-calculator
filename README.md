# Pizza Calculator

A tool for figuring out whether it's better value to buy a large pizza or small ones. You enter the size, count, and price for each option, and the app calculates the price per cm², visualizes the area difference with to-scale circles and bar charts, and tells you which option wins and by how much. Available in English and Hungarian.

## Stack

- vite
- react 19 with typescript
- react-i18next / i18next
- tailwindcss v4
- eslint + prettier
- editorconfig
- husky / lint-staged
- Inter & Fraunces variable fonts
